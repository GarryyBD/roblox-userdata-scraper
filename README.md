# Roblox user data scraper

Scrape roblox user data for fun!

## Installation

```bash
pip install -r requirements.txt
```

Fields that will be scraped:

description
created
isBanned
externalAppDisplayName
hasVerifiedBadge
id
name
displayName

## Usage

```bash
python src/main.py
```

## Environment variables

```
MONGO_URI="your mongo uri"

MAX_WORKERS=50
MAX_GENERATIONS=5
```
